# lexer
Лексер на основе генератора лексеров [flex](flex.sourceforge.net)
## Использование
lexer [-filter] file  
Принимает на вход несколько файлов, указанных в аргументах  
Выводит текст на консоль строки:  
```FILENAME("<Имя_файла>");``` - обрабатываемый файл  
```KW_<ключевое_слово>(line, start, end);``` - соответствует ключевому слову  
```Var("<имя_переменной>", line, start, end);``` - соответствует переменной  
```Op(<имя_оператора>, line, start, end);``` - соответствует оператору  
```Num(<число>, line, start, end);``` - соответствует числу  
```Semicolon(line, start, end);``` - соответствует символу ;  
```Open_bracket(line, start, end);``` - соответствует открытой скобке  
```Close_bracket(line, start, end);``` - соответствует закрытой скобке  
```Assignment(line, start, end);``` - соответствует присваиванию  
```LineComment("text", line, start, end);``` - соответствует однострочному комментарию  
```MultilineComment("text", start_line, start, end_line, end);``` - соответствует мнострочному комментарию  
```EOF("<Имя_файла>");``` - означает конец обработки файла  


```line``` - номер строки, ```start``` - номер начального символа, ```end``` - номер конечного символа  
```start_line```, ```end_line``` - номер строки начала и конца многострочного комментария соответственно  
## Файлы
```bin\``` содержит исполняемые файлы  
```source\``` содержит исходный код лексера  
```testfiles\``` содержит набор тестовых данных  
```makefile``` - можно использовать ```make``` для сборки ([mingw](http://www.mingw.org/),[Make for Windows](http://gnuwin32.sourceforge.net/packages/make.htm) )  
```test.py``` - запускает лексер с тестом из ```testfiles```
